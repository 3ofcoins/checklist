# Checklist

Execute your tasks as checklist, get clear status of what has been
done and what has not on your terminal.

## Installation

Add this line to your application's Gemfile:

    gem 'checklist'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install checklist
